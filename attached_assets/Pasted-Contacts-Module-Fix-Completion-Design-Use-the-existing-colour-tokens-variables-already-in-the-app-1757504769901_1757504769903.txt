Right panel: searchable property catalogue sourced from AICRM Contact schema (dynamic), grouped (e.g., Contact Info, Calculated, Activity, Marketing/Consent, Scoring).

Text: contains/equals/starts/ends/empty/not empty

Enum: is/is not/in/not in

Number: =, ≠, >, ≥, <, ≤, between

Date: before/after/between/is empty

Boolean: is true/false

Persist Saved Views (name, filters, sort, columns, page size).
